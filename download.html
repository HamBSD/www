HamBSD Download

Watch out during installation that you are using the sets from the installation media or from the HamBSD preview URL!
If you are not careful, you will just end up installing an OpenBSD snapshot.

A preview build is available for download. For best results, choose the installXX.fs or installXX.iso and install the file sets from the disk. If you do install the file sets via HTTP, the server hostname is irl.radiofreqs.space and the path is pub/HamBSD/preview/amd64.

  • installXX.fs [amd64]
    A disk image that can be written to a USB flash drive or similar device.
    Includes the file sets.

  • minirootXX.fs [amd64]
    The same as above, but file sets are not included. They can be pulled down from the internet or from a local disk.

  • installXX.iso [amd64]
    An ISO 9660 image that can be used to create an install CD/DVD.
    Includes the file sets.

  • cdXX.iso [amd64]
    The same as above, but file sets are not included.

  • floppyXX.fs [amd64]
    Supports some older machines that lack other booting options.
